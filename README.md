## 发布到 GitHub Pages

1. 把整个项目推到一个 GitHub 仓库
2. 在 GitHub 仓库里打开 `Settings -> Pages`
3. `Source` 选择 `Deploy from a branch`
4. 分支选 `main`
5. 文件夹选 `/ (root)`
6. 保存后等待几十秒到几分钟

发布成功后，网址通常会是：

`https://你的GitHub用户名.github.io/仓库名/`
