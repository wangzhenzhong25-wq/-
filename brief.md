# 项目简报

## 目标

增强 BB 六选二 HTML 源码的复习时间记录和错题集能力。

## 背景

当前网页已有单元、测试、星标、记忆曲线等功能，但缺少：
- 单元级“当日已复习”手动记录
- 测试错题自动进入错题集
- 错题集查看、重测和移除

## 交付物

- 修改后的 HTML 源码副本
- 项目说明与过程记录

## 范围

- 保留现有单元、测试、星标、复习逻辑
- 新增单元复习时间记录
- 新增错题集页面与专项测试入口
- 更新导入导出数据结构

## 不做什么

- 不改为桌面软件
- 不重构全部 UI 架构
- 不迁移到新框架

## 验收标准

- 用户可手动标记某单元“当日已复习”，并看到当天时间
- 错题作答后自动加入错题集
- 可查看错题集、重测错题、单独移除或清空
- 导入导出包含新数据，不破坏旧数据兼容

## 风险和注意事项

- 需要尽量复用现有状态与本地存储键，避免互相覆盖
- 需要控制改动范围，避免把现有测试与复习逻辑带坏
